## Sobre o Projeto

Sistema completo desenvolvido para o **Mercado Solidário**, com o objetivo de facilitar a doação e distribuição de alimentos e itens básicos.

### Funcionalidades Principais
- Dashboard administrativo
- Cadastro e gerenciamento de doações
- Controle de beneficiários
- Interface responsiva
- Integração com banco de dados MySQL

**Tecnologias:** PHP, JavaScript, HTML, CSS, MySQL

## Sobre o Projeto

Aplicação desktop para cadastro, consulta, edição e exclusão de livros com conexão real ao banco de dados.

### Funcionalidades
- CRUD completo (Criar, Ler, Atualizar, Deletar)
- Interface gráfica com Java Swing
- Conexão JDBC com MySQL
- Busca e listagem de livros
- Validações de campos

**Tecnologias:** Java, MySQL, JDBC
